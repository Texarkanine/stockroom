# stockroom

A local, faithful, searchable warehouse of your agentic-coding history.

Stockroom captures prompts, responses, and tool inputs from Cursor and Claude Code into a single-file DuckDB warehouse with local embeddings. Day-to-day use goes through the `sr-*` skills in your agent harness; this site is the human guide for install, troubleshooting, advanced CLI use, and contributing.
